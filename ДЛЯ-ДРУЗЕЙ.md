# 🎮 Как подключиться к Discord Clone

## 📥 Шаг 1: Скачать проект

Откройте терминал (Command Prompt или PowerShell) и выполните:

```bash
git clone https://github.com/Motiksto/discord-clone.git
cd discord-clone/client
```

**Если нет git:**
1. Скачайте ZIP: https://github.com/Motiksto/discord-clone/archive/refs/heads/main.zip
2. Распакуйте архив
3. Откройте терминал в папке `discord-clone-main/client`

---

## 🔧 Шаг 2: Установить зависимости

```bash
npm install
```

**Если нет Node.js:**
Скачайте с https://nodejs.org (версия LTS)

---

## 🌐 Шаг 3: Подключиться к серверу

Создайте файл `.env` в папке `client`:

**Windows (PowerShell):**
```powershell
echo VITE_SERVER_URL=https://real-files-spend.loca.lt > .env
```

**Windows (Command Prompt):**
```cmd
echo VITE_SERVER_URL=https://real-files-spend.loca.lt > .env
```

**macOS/Linux:**
```bash
echo "VITE_SERVER_URL=https://real-files-spend.loca.lt" > .env
```

---

## 🚀 Шаг 4: Запустить

```bash
npm run dev
```

Откройте в браузере: **http://localhost:5173**

---

## ✅ Готово!

Теперь вы подключены к серверу и можете:
- Зарегистрироваться
- Создавать чаты и группы
- Отправлять сообщения
- Использовать голосовые каналы

---

## ❓ Проблемы?

### "npm: command not found"
Установите Node.js: https://nodejs.org

### "git: command not found"
Скачайте ZIP архив вместо клонирования

### Не подключается к серверу
Проверьте, что в `.env` правильный URL: `https://real-files-spend.loca.lt`

### Порт 5173 занят
Vite автоматически выберет другой порт (например, 5174)
